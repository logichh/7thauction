# 7th Auction House

A feature-rich auction house and trading plugin for Minecraft servers. Create a thriving player-driven economy with an intuitive GUI-based marketplace.

## Key Features

### Auction House
- **Buy It Now (BIN) Listings** - Instant purchase auctions for quick transactions
- **Bid Auctions** - Traditional auction system with competitive bidding
- **Smart Search** - Find items quickly with an intuitive search system
- **Personal Auctions Manager** - Track and manage your active listings
- **Auto-Collection** - Optionally auto-deposit earnings when items sell
- **Partial Buying** - Let buyers purchase specific quantities from listings
- **Auction Announcements** - Server-wide notifications for new listings (toggle-able)

### Player Trading
- **Direct Player Trading** - Safe peer-to-peer item and money exchanges
- **Trade Requests System** - Send and accept trade offers with built-in cooldowns
- **Security Features** - Configurable limits and timeouts to prevent exploitation

### Economy & Configuration
- **Flexible Tax System** - Configurable transaction fees for economy balance
- **Custom Limits** - Set maximum auctions per player, price limits, and duration caps
- **Shop Integration** - Block specific items from auction house to protect your server shop
- **Multiple Durations** - Players can choose listing durations for bid auctions

### World Integration
- **Display Items** - Physical item displays in your world (requires Heavenly Blessing)
- **Item Showcases** - Create visual marketplaces with real item displays

### Admin Tools
- **Moderation Commands** - Expire or delete problematic auctions with reasons
- **Admin Search** - Quickly find and manage any auction
- **Blocked Items Management** - GUI-based system to manage restricted items
- **Customizable Permissions** - Fine-grained control over who can use features

### Cross-Platform Support
- **Java & Bedrock Compatible** - Full support for Bedrock players via Geyser + Floodgate
- **Optimized GUIs** - Bedrock-friendly 6x9 menus without filler items
- **PlaceholderAPI Support** - Integrate with other plugins seamlessly

## Main Commands

- `/ah` - Open the auction house
- `/ah sell <price>` - List the item you're holding as a BIN auction
- `/ah bid <price>` - List the item you're holding as a bid auction
- `/trade <player>` - Send a trade request to another player

## Fully Customizable

- Modify all messages and colors in `messages.yml`
- Configure features, limits, and durations in `config.yml`
- Customize GUI layouts in `layout.yml`
- Adjust permissions in `permissions.yml`
